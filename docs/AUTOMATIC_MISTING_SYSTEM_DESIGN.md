## 사육장 연결 구성

- 노즐은 **T형 분기** 방식으로 사육장 내 여러 방향 분사 가능
- 수중 모터 펌프를 물통에 수몰 설치 후 노즐까지 튜브 연결
- 노즐 설치 위치: 사육장 측면 또는 상단

---

## 전기 연결도

```
12V 전원 어댑터
    ├─ 12V 직결 → 다이어프램 펌프 0142 (릴레이/MOSFET 경유, 3.75A)
    └─ 5V 변환 (레귤레이터 또는 USB) → ESP32, 릴레이, DHT22

ESP32 GPIO → 릴레이(또는 MOSFET) → 펌프 12V ON/OFF
ESP32 GPIO → DHT22 데이터핀 (풀업 저항)
```

- ESP32 전원은 5V 레귤레이터 또는 USB 공급 모두 가능
- 펌프 제어: 릴레이 또는 MOSFET 모듈 선택 가능

## 시스템 동작 흐름

```
시작
 └─ ESP32 WiFi 연결 / 초기 설정
     └─ DHT22 온습도 읽기 (루프)
         ├─ 습도 < 설정값?
         │   YES → 릴레이/MOSFET ON → 펌프 작동
         │          → 5초 대기 → 릴레이 OFF
         │   NO  → 10초 대기 → 다시 읽기
         └─ 수동 명령 처리 (앱 / MQTT)
```

## 스마트홈 연동 구성

- **Node-RED** 서버 연동
  - 텔레그램 봇 알림 (습도 이상 시 푸시)
  - MQTT 브로커 연동
  - 스케줄 관리
- **모바일 앱 UI** (참고 이미지 포함)
  - 현재 온도 / 습도 표시
  - 자동 분무 임계값 설정
  - 수동 분무 버튼
  - 스케줄 설정

---

## 사용 팁 & 주의사항

- **분무 방향**: 측면 또는 위에서 아래 방향으로 설치
- **물통 위치**: 펌프보다 낮거나 같은 높이 권장 (역류 방지)
- **노즐 막힘 방지**: 증류수 사용 권장
- **자동 분무 시간**: 코드 내 설정값으로 조절
- **습도 센서 위치**: 공기 순환이 잘 되는 곳에 설치
- **전원 OFF 시**: 펌프 자동 OFF (안전)

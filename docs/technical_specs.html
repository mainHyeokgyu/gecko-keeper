GET /api/sensor/history?hours=24

분무 제어

POST /api/spray/manual
수동 분무 실행 (duration: 초)
PUT /api/spray/auto-config
자동 분무 설정 (threshold, enabled)

스케줄

GET /api/schedule/list
등록된 스케줄 목록

📨 Kafka Topic 설계

sensor.telemetry
ESP32에서 전송하는 온습도 센서 데이터
Key: device_id | Value: {temperature: 25.6, humidity: 72.3, timestamp: ...}
actuator.command
분무기 제어 명령
Key: device_id | Value: {action: "SPRAY_ON", duration: 5}
actuator.result
분무 실행 결과
Key: device_id | Value: {status: "SUCCESS", executed_at: ...}
alert.event
임계값 초과 등 경보 이벤트
Key: enclosure_id | Value: {type: "HUMIDITY_LOW", value: 45, threshold: 60}

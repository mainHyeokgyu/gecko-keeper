# 🧩 2단계: 제품 컨셉 구체화

## 🎯 목표

“우리가 만들 제품이 정확히 뭐냐?”

---

## 해야 할 일

- 핵심 기능 정의
- MVP 범위 설정
- 제외 기능 결정

---

## 산출물

### 1. 기능 명세서 (Feature List)

```
[필수]
- 원격 분무 제어
- 온습도 모니터링
- 실시간 상태 확인

[선택]
- 홈캠
- 행동 분석
- 자동 제어
```

---

### 2. MVP(Minimum Viable Product) 정의

```
MVP 범위:
- 센서 + 분무기 제어
- 웹에서 ON/OFF
- 기본 로그 확인
```

👉 여기서 욕심 줄이는게 중요

---

### 3. 사용자 플로우

```
로그인 → 사육장 선택 → 상태 확인 → 분무 실행
```

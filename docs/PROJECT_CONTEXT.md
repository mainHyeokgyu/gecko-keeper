# 🦎 파충류 스마트 사육장 플랫폼 - 프로젝트 컨텍스트

## 📋 프로젝트 요약
크레스티드게코용 스마트 사육장 자동 습도 관리 시스템 프로토타입 개발

## 🎯 현재 결정된 프로토타입 스펙

### **타겟 동물**: 크레스티드게코
- 필요 습도: 60-80%
- 특성: 습도 민감, 야행성

### **센서 정확도**
- 온도: ±1°C 미만
- 습도: ±3% 미만

### **분무 시스템**
- **확정**: 펌프+노즐 방식 (0142 압력형 다이어프램 펌프 DC 12V / 3.75A, 작동압력 4.5kg/cm² + 황동 분사 노즐 1.5mm)
- 이유: 벽면 물방울 형성 필수 (게코 수분 보충) — 초음파 진동자는 에어로졸만 생성하여 부적합

### **제어 기능** (MVP 범위)
- **수동**: 앱에서 즉시 분무 ON/OFF
- **자동**: 습도 임계값 기반 자동 분무
- **스케줄**: 시간별 자동 분무 설정

### **통신 방식**
- **확정**: WiFi 직접 연결 (ESP32 → 서버)
- 이유: 단순, 빠름, 프로토타입에 적합

## 🔄 후속작 고려사항

### **통신 시스템 확장**
- **MQTT Gateway 방식**: 확장 시 고려
  - 장점: 안정적, 다중 디바이스 관리 용이
  - 활용 시점: 다중 사육장 지원 시

## 🛠️ 기술 스택

### **하드웨어**
- MCU: ESP32 (WiFi 내장)
- 센서: DHT22 (온습도)
- 액추에이터: 0142 압력형 다이어프램 펌프 (12V / 3.75A) + T형 분기 노즐 황동 유형B (4/7mm T커넥터 + 황동 노즐 10EA), 릴레이 제어
- 전원: 12V 5A 어댑터 (DC잭 5.5×2.5mm)

### **소프트웨어**
- Embedded: Arduino/ESP-IDF
- Backend: Spring Boot + Kafka
- Frontend: React
- Database: Redis(캐시) + TimescaleDB(시계열)

## 🎨 시스템 아키텍처 (프로토타입)

```
[크레스티드게코 사육장]
        ↑
[ESP32 + 센서 + 다이어프램 펌프]
        ↓ WiFi
    [백엔드 서버]
        ↓
[웹/모바일 앱]
```
